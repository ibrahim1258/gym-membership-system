# Gym Membership System

A simple gym membership web app.

## Files

- `index.html`: app layout
- `assets/css/styles.css`: design and responsive styling
- `assets/js/app.js`: login, members, subscriptions, and backup logic

## Current Status

This version is a static website that stores data in the browser with IndexedDB.
It is good as a prototype, but data does not sync automatically between devices.

## What It Can Do

- Admin login
- Add, edit, and delete members
- Member login
- Show remaining subscription days
- Send suggestions
- Export and import backup data
